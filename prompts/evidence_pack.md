# Prompt: Evidence Pack

<!-- TODO: Write this prompt -->

Given a selected forecast question, build an evidence pack.

The evidence pack should include:

- Forecast question (precise, unambiguous)
- Resolution criteria (how and when will this be decided?)
- Deadline
- Relevant evidence items, each with: source, date, summary, and direction (supports / contradicts / neutral)
- Conflicting information and how to weigh it
- Missing evidence that would significantly change the estimate
- Prior forecast if one exists

The evidence pack should be readable by both a human reviewer and an LLM in a subsequent step.
