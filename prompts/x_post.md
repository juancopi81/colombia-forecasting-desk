Given a reviewed forecast draft, write a final X post.

Guidelines:

- Under 280 characters
- State the question, the probability, and one key reason
- Cite at least one source by name (not a URL unless space allows)
- Do not give advice of any kind
- Do not use sensationalist language
- Write in a direct, factual tone
- Include a thread reply if additional context is needed

Example format:
"[Question]? We estimate [X]% probability by [date]. Key signal: [source] reported [evidence]. [Caveat or update trigger]."
