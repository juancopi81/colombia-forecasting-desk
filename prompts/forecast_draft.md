# Prompt: Forecast Draft

<!-- TODO: Write this prompt -->

Given a completed evidence pack, produce a forecast draft.

The draft should include:

- Forecast question
- Probability estimate (0–100%)
- Confidence level (low / medium / high)
- Summary of reasoning
- Key evidence supporting the estimate
- Strongest counterargument
- What would change the estimate significantly
- Draft X post (under 280 characters, source-backed, no advice)

Do not claim certainty. Do not give investment, trading, or betting advice. Flag any ethical concerns before producing the draft.
