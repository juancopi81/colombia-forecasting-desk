# Prompt: Question Selection

<!-- TODO: Write this prompt -->

Given the daily brief, propose a ranked list of candidate forecastable questions.

For each question, score it on:

- Interest: how important is this to Colombian politics, economy, or society?
- Forecastability: can a probability be meaningfully assigned?
- Evidence availability: is there enough public evidence to reason about it?
- Freshness: is this question timely and unresolved?
- Risk: are there reputational, legal, or ethical concerns?

Select the top 1–3 questions for deeper research. Explain the selection briefly.
